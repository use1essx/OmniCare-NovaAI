📋 How to Fix Microphone Issues:

  1. Look for the 🎤 icon in your browser's address bar
  2. Click it and select "Allow" for microphone access
  3. If no icon appears:
    • Go to browser Settings → Privacy & Security → Site Settings
    • Find "Microphone" and allow access for this site
  4. Refresh this page and try again
