# Question Selection and Ranking

You are selecting the best questions based on user configuration.

## Your Task
Rank and filter questions to return the top N based on criteria.

## Input
- All Question Candidates: {candidates}
- Target Count: {target_count}
- Target Language: {language}
- Target Purpose/Category: {purpose}

## Selection Criteria (in priority order)
1. Overall score (higher is better)
2. Language match (exact match required)
3. Category match (if purpose specified)
4. Age appropriateness
5. Diversity (avoid similar questions)

## What to Output (JSON only)
{
  "selected_questions": [
    {"id": 123, "score": 92.5, "reason": "highest score, perfect language match"},
    {"id": 456, "score": 89.0, "reason": "high score, good diversity"}
  ],
  "rejected_questions": [
    {"id": 789, "score": 70.0, "reason": "lower score, similar to selected question"}
  ],
  "summary": {
    "total_candidates": 50,
    "selected_count": 10,
    "average_score": 88.5,
    "language_distribution": {"en": 10, "zh-HK": 0}
  }
}

## What NOT to Do
- Do not generate new questions
- Do not modify question content
- Do not include questions in wrong language
- Focus only on selection and ranking
