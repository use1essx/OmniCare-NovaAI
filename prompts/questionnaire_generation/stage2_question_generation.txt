## Strict Fidelity Requirements

### 85-90% Rule

- **85-90%** of questions MUST be directly from the document
- **10-15%** can be clarifying questions ONLY if absolutely necessary
- **0%** should be from your general knowledge

### What Counts as "From Document"

✅ **Acceptable**:
- Exact question from document
- Translated version of document question
- Reformatted for digital use (e.g., converting table row to question format)

❌ **NOT Acceptable**:
- Questions you think would be helpful
- Questions from your knowledge of the topic
- "Improved" versions of document questions
- Additional questions to "fill gaps"

---

## Quality Requirements

1. **Exact Wording**: Use the exact wording from the document (or accurate translation)
2. **Preserve Structure**: Keep original categories, domains, and organization
3. **Maintain Scoring**: Use the exact scoring system from the document
4. **Bilingual Accuracy**: If document is bilingual, preserve both languages
5. **Assessment Method**: Correctly identify if it's interview, observation, or hands-on
6. **Materials**: Note any picture cards, props, or materials mentioned
7. **Age Appropriateness**: Use the age range specified in the document

---

## What NOT to Do

❌ **Do NOT** generate questions from your general knowledge about child development
❌ **Do NOT** add questions that aren't in the source document
❌ **Do NOT** create variations or alternatives to document questions
❌ **Do NOT** reorganize the structure (keep original domains/categories)
❌ **Do NOT** modify the scoring system
❌ **Do NOT** paraphrase unless translating to target language
❌ **Do NOT** add "helpful" questions to improve the assessment
❌ **Do NOT** generate questions just to reach a target count

## Special Cases

### If Document Has Tables

Extract each row as a separate question item, preserving:
- Row number/identifier
- Question text
- Response options
- Scoring criteria

### If Document Has Picture Cards

Note the picture card reference (e.g., "Card 5", "圖卡9") and describe what it shows based on the document.

### If Document Has Multiple Languages

Preserve all language versions exactly as they appear in the document.

### If Document Has Scoring Tables

Extract the complete scoring framework:
- Point values
- Pass/fail thresholds
- Domain-specific cutoffs
- Interpretation guidelines
