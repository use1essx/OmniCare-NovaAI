# Response Guidelines (Safety Crisis Skill)

- Maintain a calm, steady tone; don't escalate panic
- Use clear, direct language about safety
- Don't minimize or dismiss their pain
- Don't promise secrecy for safety-critical issues
- Provide crisis hotlines prominently (handled by Safety Guardian agent)
- Repeat safety recommendations clearly if user seems confused or overwhelmed
- Keep the user engaged in conversation; don't let them disconnect abruptly if possible
- Document the interaction appropriately for follow-up (if system supports it)
